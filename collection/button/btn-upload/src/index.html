• draggable="true" — для вкл возможности переносить (тянуть) элементы
  • dragenter — перемещаемый элемент входит в принимающий объект
  • dragover — перемещаемый элемент проходит над принимающим объектом
  • dragleave — перемещаемый элемент покидает принимающий объект
  • drop — пользователь отпускает перемещаемый элемент
